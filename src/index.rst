=========
 OpenCOR
=========

`OpenCOR <http://www.opencor.ws/>`__ is a cross-platform modelling environment, which is aimed at organising, editing, simulating and analysing CellML files on `Windows <https://en.wikipedia.org/wiki/Microsoft_Windows>`__, `Linux <https://en.wikipedia.org/wiki/Linux>`__ and `macOS <https://en.wikipedia.org/wiki/MacOS>`__.
It can be downloaded `here <http://www.opencor.ws/downloads/index.html>`__.

Various information about OpenCOR and its use can be found in the following pages:
